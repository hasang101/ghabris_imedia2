Microsoft Office

Microsoft office has many applications that are helpful to use in your everyday life. Listed below are some of their application descriptions.

Microsoft Office Applications Descriptions /p>

Word: A word processing application that allows you to create a variety of documents, including letters, resumes, and more.

Power Point: A presentation program that allows you to create dynamic slide presentations.

Excel: An electronic spreadsheet is a computer software program that is used for storing, organizing and manipulating data.

These applications are always being updated to keep up with today's growing technology.
